# Bot do Zap 2.0

Bot para envio de mensagens automáticas no WhatsApp com PyAutoGui<br/>
Atualmente, funciona apenas para monitores com resolução 1920 x 1080

# Instalação das dependências (CMD)

pip install -r requirements.txt
